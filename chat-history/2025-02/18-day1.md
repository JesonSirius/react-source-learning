### 4. 源码管理设置
Q: 你能随时访问到react的源码吗，需要我把源码放到这个仓库吗
A: 建议使用git submodule管理React源码：

**配置步骤：**
```bash
git submodule add https://github.com/facebook/react.git react-source
git submodule update --init --recursive
```

**主要优势：**
- 源码独立管理
- 便于版本切换
- 不占用主仓库空间
- 支持源码调试
